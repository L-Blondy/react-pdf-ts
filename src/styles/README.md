# buttons and links

**Common Modifiers**:

- .disabled
- .medium
- .large
- .primary - *red bg*
- .success - *green bg*
- .light   - *light grey bg*

*******

## **.btn-minimal**

- No borders
- Blue circle around on hover + Blue bg

**Modifiers**:

- *no specific modifier yet*

## **.btn-primary**

- Red background / light font
- large icon by default
- brightens on hover

**Modifiers**:

- .icon-small
- .icon-large

*******

## **.btn-square**

- use for icons
- icons need to be squared
- the svg base size cannot be shrinked properly

**Modifiers**:

- .rotate-180
